전산현황

 

- 구매 : 타업체 ERP 사용

- 스마트공장으로 MES 도입

 

✓ MES 구축 이후 연결할 시스템에 대한 개발 능력이 없음(ERP, 회계, 인사, 표준원가 등)

✓ 성공사례 소개도 일부 업체만 제공하여 정보를 왜곡

✓ 실시간 정보에 대한 가치를 모르며 활용방안도 없음

✓ 빅데이터의 개념과 분석 방법 모름

✓ 외적 기능과 화면의 화려함만 고려

✓ 도입기업의 스마트공장에 대한 이해도 낮음

✓ 중소벤처기업부의 성공사례 견학의 화려한 모습이 성공으로 착각

✓ 기 구축 ERP와 연동이 불가능한데 해당 부분 점검 안함

✓ 실시간 데이터 입력보다 현장에 설치되는 기기에 관심 많음

✓ 쇠퇴하는 프로그램 언어를 구분 못함

✓ 오랜 실패와 경험을 통해 도입기업에 실패를 적게할 수 있는 방법을 알려줄 전문가 부족

✓ 도입기업과 동일하게 화려한 문서와 메뉴 구성에만 관심을 갖음

✓ 단계별 구축 이후 필요한 기술에 대하여 표면적 모습만 알고 도입기업에 컨설팅

✓ 수기로 적은 것을 저녁에 엑셀에 넣는 것과 크게 다르지 않음

✓ 작업 중 입력을 하러 한참 걸어가야 하는 동선 문제

✓ 키오스크의 화려함만 있으며 현장 작업자의 입력에는 관심 없는 시스템

✓ 실시간 입력을 하더라도 방향성이 없어 얻을 수 있는 효과가 전혀 없음

✓ 테이블 설계가 실시간 데이터 입력으로 얻을 수 있 효과가 없도록 설계됨
