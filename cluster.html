운영중인 시스템의 하드웨어 및 서비스의 장애 발생 시 이를 자동으로 인지하여 대기중인 시스템에서 연속적인 서비스를 수행 할 수 있는 HA 솔루션입니다.


Cluster 특장점

  • 서비스의 연속성 보장

    1서비스, 프로세스 단위의 리소스 감지

      - 다양한 어플리케이션 감지 및 구동 방식 지원

    2SQL, WAS 단위의 리소스 감지

      - DBMS DB 상태에 대한 감지

      - IIS내 WAS 상태에 대한 감지

    3HW 리소스 감지

      - Network, Split 상태 감지

      - HDD, NIC 및 디스크 용량 감지

    4사용자 정의 리소스 감지

      - 사용자 정의의 Check 감지

  • 데이터 안정성 지원

    1DBMS 동기화 지원

      - MSSQL 트랜잭션 동기화 지원

      - MSSQL 동기화 데이터에 대한 무결성 검증 지원

    2File 동기화 지원

      - 커널 I/O 이벤트 감지를 통한 폴더 및 파일 단위의 실시간 동기화 지원

    3Block 복제 지원

      - 볼륨 동기화에 대한 구성 및 연동 지원

  • 데이터 백업 및 DR 구성

    1백업 및 복구 지원

      - MSSQL 전체/차등/증분 백업 지원

      - 폴더 및 파일 단위의 백업 지원

      - 백업된 내용의 복구 기능 지원

    2DR 구성 지원

      - DR 구성에 따른 Active 시스템의 데이터 복제 기능 제공

    3설정 복구 기능

      - 이중화 구성 설정을 백업/복구 기능 제공



Cluster 기능

  • 다양한 Application 및 구성 지원

    1Database, Web, ERP등 다양한 Application 지원

    2Shared Storage를 이용한 Active-Standby 구성 지원

    3SAN, DAS, Multi Path I/O 운영환경 지원
